Step-by-step setup guide

Getting your YouTube, Gemini, and ChatGPT API keys — and getting started with YTInsight.

Step 1
Create Your Account
Click "Login to Dashboard" in the nav. Enter your email and password to sign in. Your account is secured via Firebase Authentication — no registration form needed.
Step 2
Get Your YouTube API Key
Go to Google Cloud Console → Create a project → Enable "YouTube Data API v3" → Create Credentials → API Key.
console.cloud.google.com

Enable: YouTube Data API v3 → Credentials → Create API Key

Step 3
Get Gemini API Key (Free)
Gemini has a generous free tier — recommended for most users. Go to Google AI Studio and generate a key in seconds.
aistudio.google.com/app/apikey

Click "Create API Key" → copy and save. Free tier: 60 requests/min.

Step 4
Get ChatGPT API Key (OpenAI)
If you prefer GPT-4 for SEO generation, go to OpenAI Platform and add billing.
platform.openai.com/api-keys

Create key → Add billing → Recommended model: gpt-4o-mini for cost efficiency.

Step 5
Add Keys in Dashboard
Login → Dashboard → API Keys panel. Paste your YouTube key, then choose Gemini or ChatGPT and paste that key. Keys are saved securely to Firebase RTDB under your account.
Step 6
Analyze & Generate
Paste any YouTube channel URL or video URL in the top bar → click Analyze. Select a video from the grid → click Generate SEO or Generate Thumbnail. Your chosen AI engine handles the rest.
